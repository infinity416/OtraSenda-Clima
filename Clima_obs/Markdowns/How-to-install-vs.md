# Cómo instalar Visual Studio Code para leer archivos Markdown

---

## ¿Qué es un archivo Markdown (.md)?

Es un archivo de texto con formato enriquecido: títulos, listas, tablas y bloques 
de código. Para leerlo de forma cómoda necesitas Visual Studio Code (VS Code), 
un programa gratuito.

---

## Paso 1: Descargar Visual Studio Code

1. Abre tu navegador y entra a: https://code.visualstudio.com
2. Haz clic en el botón azul **Download for Windows**
3. Se descargará un archivo llamado `VSCodeSetup-x64-....exe`
4. Ábrelo y sigue los pasos del instalador (puedes dejar todas las opciones 
   por defecto y solo hacer clic en **Siguiente** hasta terminar)

---

## Paso 2: Abrir Visual Studio Code

1. Busca **Visual Studio Code** en el menú de inicio de Windows
2. Ábrelo — verás una pantalla de bienvenida en inglés, eso es normal

---

## Paso 3: Instalar la extensión para leer Markdown

1. En la barra lateral izquierda, haz clic en el ícono de **cuatro cuadros** 
   (Extensions)
2. En el buscador que aparece escribe: `Markdown Preview Enhanced`
3. Haz clic en el resultado que dice exactamente **Markdown Preview Enhanced**
   (autor: Yiyi Wang)
4. Haz clic en el botón azul **Install**
5. Espera unos segundos hasta que diga **Installed**

---

## Paso 4: Abrir y leer un archivo .md

1. En VS Code, ve al menú **File → Open File...**
2. Busca el archivo `.md` en tu computadora y ábrelo
3. Verás el texto del archivo en la pantalla
4. Para verlo con formato (títulos, tablas, etc.), haz clic derecho sobre 
   el archivo y selecciona **Open Preview to the Side**
5. Se abrirá una vista a la derecha con el documento formateado y listo 
   para leer

---

## ¿Problemas frecuentes?

| Problema | Solución |
|----------|----------|
| El instalador pide permisos de administrador | Haz clic en "Sí" para continuar |
| No encuentro el archivo `.md` | Busca en la carpeta donde te lo compartieron |
| La vista previa no abre | Asegúrate de que la extensión esté instalada correctamente (paso 3) |

---

> 💡 **Tip:** Una vez abierta la vista previa, puedes agrandar esa pantalla 
> arrastrando el borde hacia la izquierda para leer más cómodamente.
